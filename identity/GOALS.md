# GOALS

**What specific, measurable objectives are you working toward?**

## Why This Matters

Your PILOT agent uses goals to:
- Prioritize your work
- Suggest relevant learning
- Track progress over time
- Celebrate completions
- Remind you of objectives

---

## Tips

- **Be specific**: "Get better at K8s" → "Complete CKA certification"
- **Make measurable**: "Improve performance" → "Reduce API latency to <200ms"
- **Set deadlines**: "Learn Go" → "Build 3 Go microservices by March"
- **Review regularly**: Update this file weekly/monthly
- **Celebrate wins**: Mark completed goals with [x] and date
